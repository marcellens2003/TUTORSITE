Change password when current password is known

Browser pages cannot reset an unknown Firebase Auth password for another user. This section works only when you know the current password.

Forgotten password helper

Because tutors sign in with synthetic emails like pgt0420@tutors.ucc.local, normal password-reset emails are not useful. For a forgotten password, delete the Auth user in Firebase Console, then recreate the login here with a new password.
